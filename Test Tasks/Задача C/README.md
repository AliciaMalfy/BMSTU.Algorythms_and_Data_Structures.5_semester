# Задача
```
Во входном файле задаются несколько целых чисел. В выходной файл напечатайте сумму этих
чисел по модулю 256. Числа задаются по одному в строке. Пробельные символы перед числом
и после него отсутствуют. Имена входного и выходного файла задаются через аргументы командной
строки (первый и второй соответственно).
```
